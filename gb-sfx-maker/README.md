# GB SFX Maker

Browser-based Game Boy wave-channel sound effect editor.

## Features

- Eleven starter presets inspired by classic JRPG battle sound effects
- GB wave memory editor with 32 4-bit samples
- Pitch bend, volume fade, vibrato, retrigger, and noise-mix controls
- Repeat count and repeat gap controls for multi-hit sword attacks
- Audition in the browser with Web Audio
- Save customized effects under new names in local storage
- Export and import JSON
- Render the current effect to WAV

## Commands

```sh
npm install
npm run dev
npm run build
```
